# Credit Card Fraud Monitoring & Risk Analytics Dashboard

## 📊 Project Overview

This project analyzes credit card transaction data to identify fraud patterns and provide insights using Power BI.
The dashboard helps monitor fraudulent activities, high-risk transaction categories, and geographical fraud hotspots.

## 🎯 Objectives

* Detect patterns in fraudulent transactions
* Identify high-risk merchant categories
* Analyze fraud trends over time
* Visualize fraud distribution across regions

## 🛠 Tools Used

* Power BI
* DAX
* Power Query
* Data Visualization

## 📈 Dashboard Features

* Executive Summary of transactions and fraud rate
* Fraud Risk Analysis by category, merchant, and gender
* Geographic fraud insights
* Interactive filters and slicers

## 📂 Dataset

Credit card transaction dataset containing transaction amount, merchant category, location, and fraud indicator.

Dataset Link: https://drive.google.com/drive/folders/1BfXQSDmd_XH-3tJmB10EDqvPw9ub0yK4
